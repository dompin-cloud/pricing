Printers

Hourly cost = electricity (avg watts × your $/kWh) + depreciation (purchase price ÷ lifetime hours) + maintenance $/hr (nozzles, plates, belts). Lifetime of ~5,000 print-hours is a common assumption — adjust to taste.

Filaments

Cost per gram = price per kg ÷ 1000. The “Legacy” entry matches your old sheet’s flat $0.05/g so imported products price similarly until you assign real filaments.

Global Settings

Your last Xcel bill: $268.95 ÷ 1,678 kWh = $0.160/kWh effective (includes fixed charges). Failure rate adds a % of material + electricity to cover failed prints. Markup default matches your old sheet’s 50%.

Cloud Sync (GitHub)

Not connected. To set up: GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token. Set Repository access to only your data repo, and under Permissions give Contents: Read and write. Set a long expiration. Paste the token above on each device — it's stored only in this browser, never in the public app.

Backup

Data saves automatically in this browser. Export a JSON backup occasionally, or to move to another computer.
